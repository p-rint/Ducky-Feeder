Made using blender and godot by Kenny!!!
Game has bread throwing and fat (or to be fat) ducks that want food!!!!
You can run it by going to the [itch.io page](https://itchiback.itch.io/ducky-feeder), pressing the "Run quack/quack" button, and playing!!

[Controls]

WASD -- Move

Click  (Left mouse) -- Throw bread!!!


Throw bread at the ducks till the pop in satisfaction!!!
